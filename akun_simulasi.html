✨ Fitur Terbaru (Versi 3.1)

🔔

Notifikasi Bell Real-Time

Dropdown notifikasi di semua layout (Admin/User/Sopir). Auto-refresh 30 detik, badge unread merah, konten berbeda per peran.

📋

Laporan Maintenance Beruntun

Admin wajib set nota 'Catatan Teknik' sebelum Bus berstatus Perawatan dan Sopir diwajibkan input 'Kondisi Harian', bersatu di Riwayat Panel Armada.

⚡

Auto-Finish Trip & Delete Protect

Selain bus tiba di tujuan hapus otomatis penumpang, armada yang tidak aktif mutlak ditahan untuk dihapus layernya.

♿

Inclusive UI & WCAG 2.1 Focus

Toolbar ♿ (font size, kontras tinggi, reduced motion) plus Purwarupa Pemesanan "Smart Blocking" khusus penyandang disabilitas.

🚨

Error Pages + Testing Dashboard

Halaman error kustom (401,403,404,419,500,503) berdesain Kampus Non-Merdeka. Dashboard uji di /test-error/ hanya aktif di environment lokal.

🗺️

Dual-Route + Admin Sync Grid

Maks 2 bus aktif per rute pada Main Live Map. Serta di Daftar Bus Admin berflashing label Trip Status tanpa polling reload dari admin.

⏲️

Auto-Cancel 15s & Driver Revive

Proteksi 15 detik Geofencing khusus kursi reguler. Mendukung Driver Override: Sopir menekan "Hadir" seketika di-revive kembali!
